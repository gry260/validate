JavaScript Form Validation with Boostrap: Quick and Easy

  • The script has a catalog of almost all the common validation types built-in.
  • The idea is to create a set of “validation descriptors” associated with each element in a form. The “validation descriptor” is nothing but a string specifying the type of validation to be performed.
  • Each field in the form can have zero one or more validations. For example, you can have an input field that should only contains integers, should be less than 25.

Using the form validation script

Call the Modules: $("input).formValidate();



input 'data-type="email"' into the attribute

input 'data-type="integer"' into the attribute

input 'data-type="double"' into the attribute

input data-chars='[",", ":"]' into the attribute. Only Allow comma and semicolon as special characters. You can specify what special characters you want to allow

input minlength='5' into the attribute. Minimum 5 characters

input maxlength='5' into the attribute. Maximum 5 characters

input data-regex='^[0-9]{4}$' into the attribute.

input data-msg='This specific field can only contains digits.' into the attribute.
